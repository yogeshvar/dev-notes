### AlgoExpert 6: Linked List

#### Singly Linked List

- Access Head => `O(1)`
- Access Tail => `O(n)`
- Access Middle => `O(n)`
- Insert/Remove Head => `O(1)`
- Insert/Remove Tail => `O(n)` to access + `O(1)`
- Insert/Remove Middle => `O(n)` to access + `O(1)`
- Search => `O(n)`

#### Doubly Linked List

- Access Head => `O(1)`
- Access Tail => `O(1)`
- Access Middle => `O(n)`
- Insert/Remove Head => `O(1)`
- Insert/Remove Tail => `O(1)`
- Insert/Remove Middle => `O(n)` to access + `O(1)`
- Search => `O(n)`
